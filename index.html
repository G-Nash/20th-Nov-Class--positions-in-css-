20th Nov Class (Positions in Css)

Positions in Css:

  1. Static
    • Default position of any element
  2. Absolute
    • position:absolute
    • whenever we give absolute position to an element, it goes to the viewport (ancestor) (default screen border) (it overrides the other magin properties)
    • top:px, left:px, right:px, bottom:px
      • changes the length from the view port
    • z-index:1 (higher value will be the first priority when overlapping)
  3. Relative
    • position:relative
    • element's position is controlled with it's normal position.
  4. Sticky
  5. Fixed

Properties of the positions:

  • top
  • left
  • right
  • bottom
  • we can't give both top/bottom and right/left value to a single element, the value that's latest will override the previous value. (top and left take the priority than others)
